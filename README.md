## Features

- React.js & Gatsby.js V2
- Markdown 写作
- 支持草稿
- Styled Components 💅
- 分类
- 标签
- 分页
- 支持 Offline
- PWA
- Typography.js
- SEO
  - Sitemap
  - RSS
  - Schema.org JSONLD(谷歌)
  - OpenGraph Tags（og 元标签）
- 自动化生成各种 Favicons
- 渐进式加载图片

## 源码使用方式

### 前置条件

- Node >= v10.0
- 全局安装 `npm i -g gatsby-cli`

### 本地开发

> 推荐使用 yarn

`yarn install && yarn dev`

### 查看效果

浏览器打开：**localhost:8000**
